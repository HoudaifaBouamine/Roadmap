Chapter 01 : Drawing Point

  • This chapter will be on the terminal, so you need only to know programming basics. The examples will be on C++, but you can use any language to achieve the same thing.

  • Problem 1: Building Screen Struct

    To show our graphics on the terminal we need to build our Screen, the screen have Rows and Colms number that represent the width and the height of the screen by the size of one character on the terminal.

  • Problem 2: Building Point Struct

    We will name our Point class stPoint to avoid confusion, and st will be the prefix of all our structs (If you are using languages like C#, do not use struct because it is a Value Type, and you cannot use pointers to get its address, use class instead).

    The stPoint must contain:

    • Members: X and Y coordinates, let it be integers.
    • Functions: draw, move, rotate.

    XIs the horizontally position of the point on the screen (the distance between the left edge of the screen and the point). Y is the vertical position of the point on the screen (the distance between the up edge of the screen and the point).

    Drawing the point will be on the terminal, and how you do this will depend on you. Move function must have 2 parameters dx and dy. If you hate math, donâ€™t worry, this just means that you will change the position of the point by dx on X axis (horizontally) and by dy on Y axis (vertically). The Rotation is the hardest part of this chapter; the rotate function gets 2 parameters, the angle of the rotation and the center of rotation point (the point around which it rotates).

    To implement the function void rotate(float angle, stPoint center), you must have a good intuition of Trigonometry and this is a good resource for that.

    Try to solve this by yourself than see the solution.
